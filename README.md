## Epstein Files — Open Investigation

When the Epstein case documents were released, they came as thousands of scanned pages — unstructured, noisy, and nearly impossible to search. We built the tools to change that.

We processed the entire document release into a searchable, structured dataset and created an open-source system that lets anyone — journalists, investigators, researchers — ask questions and get cited, page-level answers from the source materials.

- 🔍 **[Epstein Chat](https://github.com/ishumilin/epstein-chat)** – Ask questions, get answers with verifiable citations from the original documents
- 📦 **[Early Release Dataset](https://huggingface.co/datasets/ishumilin/epstein-files-ocr-datasets-1-8-early-release)** – Initial curated release for early community access
- 📄 **[Complete OCR Dataset](https://huggingface.co/datasets/ishumilin/epstein-files-ocr-complete)** – 1.38 million pages, fully processed

We welcome journalists from all over the world to dig into it.
